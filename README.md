# Image Merge to PDF (React + Tailwind)

เว็บสำหรับรวมรูปหลายไฟล์ให้เป็น PDF ไฟล์เดียว โดยสามารถจัดลำดับรูปก่อนสร้างไฟล์ได้

## Features

- อัปโหลดรูปได้หลายไฟล์
- จัดลำดับรูปได้ด้วยการลากวาง หรือปุ่มขึ้น/ลง
- ลบรูปเฉพาะรายการ หรือเคลียร์ทั้งหมด
- Export เป็น PDF 1 ไฟล์ตามลำดับที่จัดไว้
- UI โทนเขียวอ่อน-ขาว ด้วย Tailwind CSS

## Tech Stack

- React (Vite)
- Tailwind CSS
- jsPDF

## Run

```bash
npm install
npm run dev
```

เปิดที่ `http://localhost:5173`

## Build

```bash
npm run build
```
